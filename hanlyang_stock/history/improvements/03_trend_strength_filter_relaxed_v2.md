# 📈 추세 강도 필터 추가 완화 (v2)

## 📅 수정 일자
- 2025년 1월 (추가 완화)

## 🎯 주요 변경 사항

### 1. **조건부 완화 (AND → OR)** ⭐ 가장 중요
- 기존: 4개 조건 모두 만족 필요
- 변경: **4개 중 3개만 만족**해도 통과
- 효과: 필터 통과율 대폭 상승 예상

### 2. **양봉 크기 추가 완화**
- 기존: 1.0% 이상
- 변경: **0.5% 이상**
- 근거: 한국 시장의 일일 변동폭이 크지 않음

### 3. **최소 거래대금 완화**
- 기존: 3억원
- 변경: **1억원**
- 효과: 중소형주도 선택 가능

### 4. **최소 기술점수 완화**
- 기존: 0.6
- 변경: **0.5**
- 효과: 더 많은 종목이 최종 선정 단계 진입

## 📊 예상 효과

### 변경 전 (v1 완화 후)
- 한달 거래 횟수: 약 4회
- 일평균 선정 종목: 0.2개

### 변경 후 (v2 완화 후) 예상
- 한달 거래 횟수: 10-15회
- 일평균 선정 종목: 0.5-0.8개
- 승률: 약간 하락 가능 (50-55%)
- 전체 수익률: 개선 예상 (거래 기회 증가)

## 💡 추가 개선 가능 옵션

아직도 거래가 부족하다면:

1. **3개 중 2개 충족으로 완화**
   ```python
   if score >= 2:  # 현재 3 → 2로 변경
   ```

2. **거래량 조건 추가 완화**
   - 1.2배 → 1.1배로 하향

3. **RSI 범위 추가 확대**
   - 30-50 → 25-60으로 확대

4. **지지선 거리 추가 완화**
   - 5% → 7%로 확대
