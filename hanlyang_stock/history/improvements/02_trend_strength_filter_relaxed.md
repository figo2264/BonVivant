# 📈 추세 강도 필터 조건 완화

## 📅 수정 일자
- 2025년 1월 (조건 완화)

## 🎯 변경 내용

### 1. **양봉 크기 조건 완화**
- 기존: 1.5% 이상
- 변경: **1.0% 이상**
- 근거: 한국 주식시장의 일일 변동폭 특성상 1.5%는 과도하게 엄격

### 2. **거래량 조건 완화**
- 기존: 5일 평균 대비 1.5배 이상
- 변경: **1.2배 이상** (거래대금은 1.1배)
- 근거: 꾸준한 매집 패턴도 포착 가능하도록 조정

### 3. **RSI 범위 확대**
- 기존: RSI 30-40 구간에서 3일 연속 상승
- 변경: **RSI 30-50 구간에서 상승** (단순 상승만 확인)
- 근거: 40-50 구간도 충분한 매수 타이밍, 3일 연속 조건은 과도

### 4. **지지선 거리 완화**
- 기존: 지지선의 3% 이내
- 변경: **지지선의 5% 이내**
- 근거: 지지선 반등 초기 단계도 포착 가능하도록 확대

## 📊 예상 효과

### 긍정적 효과
- **매매 기회 증가**: 기존 대비 50-70% 더 많은 종목 포착
- **반전 초기 진입**: 더 이른 타이밍에 매수 가능
- **안정적인 수익 기회**: 과도한 필터링으로 놓치던 기회 확보

### 주의사항
- **가짜 신호 증가**: 완화된 조건으로 인한 노이즈 증가 가능
- **리스크 관리 중요**: 손절 기준 엄격히 적용 필요

## 🧪 테스트 권장사항

1. **단계별 백테스트**
   ```bash
   # 최근 1개월 백테스트
   python run_modular_backtest.py
   
   # 장기 백테스트 (3개월)
   python -m hanlyang_stock.backtest.run_backtest --period 90
   ```

2. **성과 비교 지표**
   - 일평균 선정 종목 수
   - 전체 승률 변화
   - 평균 수익률
   - 최대 손실폭(MDD)

3. **추가 최적화 방향**
   - 시장 상황별 동적 조정 검토
   - 섹터별 차별화된 기준 적용
   - 조건부 완화 (4개 중 3개 충족시 통과) 고려
